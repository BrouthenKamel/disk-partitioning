# Disk and partitions info

## Instructions

1. **Clone the Repository**
    ```
    git clone https://github.com/BrouthenKamel/disk-partitioning
    ```

2. **Change File Permissions**
    ```
    chmod +x run.sh
    ```

3. **Run the Script**
    ```
    ./run.sh
    ```

Follow these instructions to set up and run the project.
